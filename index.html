Proyectos

Base de datos de una urbanización para control de entrada

Este proyecto utilizó MySQL con lenguaje SQL y se crearon PROCEDURES con ayuda de Python y las siguientes librerías: "pymysql, python-dotenv, tabulate, colorama, qrcode, pillow". El objetivo principal del mismo es tener registros de las personas que entran y salen con sus cédulas y, si entran en vehículo, sus placas vehiculares, además de gestionar también las entradas automatizadas para los residentes al día con sus alícuotas.

App de cine

Esta app fue principalmente para demostrar conocimientos de Programación Orientada a Objetos como polimorfismo, herencia y encapsulamiento. Se hizo uso de Java y cuenta con interfaz gráfica diseñada para dispositivos Android desde Android Studio.

Gestionador de espacios en la ESPOL

Este proyecto demuestra conocimientos de Programación Orientada a Objetos. Es una app de consola que cuenta con tres perfiles: Administrador, Estudiante y Profesor, para reservar espacios como aulas, auditorios, etc.
